# Headless Commerce App

A modern, headless e-commerce application built with React, Vite, and Tailwind CSS. using AI for product processing.

## Features

-   **Public Storefront**: Responsive product grid, smart filtering, and AI chatbot.
-   **Admin Dashboard**: Product management with AI-powered upload and status controls.
-   **Tech Stack**: React, Vite, Tailwind CSS, Framer Motion, Lucide React.

## Getting Started

### Prerequisites

-   Node.js (v18 or higher)
-   npm or yarn

### Installation

1.  Clone the repository:
    ```bash
    git clone <repository-url>
    ```
2.  Navigate to the project directory:
    ```bash
    cd headless-commerce-app
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```

### Usage

Start the development server:

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view the app in your browser.

### Build

To build the project for production:

```bash
npm run build
```

The build artifacts will be stored in the `dist/` directory.
